kernel build.

# SYNOPSIS

## GENERAL USE

**ksymtypes** [GENERAL-OPTION...] COMMAND [COMMAND-OPTION...]

## COMMANDS

**ksymtypes** **consolidate** [CONSOLIDATE-OPTION...] PATH

**ksymtypes** **compare** [COMPARE-OPTION...] PATH1 PATH2
